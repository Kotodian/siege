# Siege Desktop (Tauri + Rust) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port Siege from a Next.js full-stack web app to a Tauri v2 desktop app with Rust backend (axum + rusqlite), targeting macOS and Linux.

**Architecture:** Tauri window loads statically-exported Next.js pages. An embedded axum HTTP server in the Rust backend replicates all 63 API routes. Frontend `fetch()` calls are redirected to `http://127.0.0.1:{PORT}` via a shared `API_BASE` variable. No Node.js runtime in the final app.

**Tech Stack:** Tauri v2, Rust (axum, tokio, rusqlite, reqwest, serde), Next.js static export, TypeScript/React

#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]

use std::net::TcpListener;
use tauri::Manager;

mod state;
mod db;
mod routes;

fn find_available_port() -> u16 {
    let listener = TcpListener::bind("127.0.0.1:0").expect("Failed to bind");
    listener.local_addr().unwrap().port()
}

#[tokio::main]
async fn main() {
    let port = find_available_port();
    let db_pool = db::init_db().expect("Failed to initialize database");
    let app_state = state::AppState::new(db_pool);

    // Start axum server in background
    let state_clone = app_state.clone();
    tokio::spawn(async move {
        let app = routes::create_router(state_clone);
        let listener = tokio::net::TcpListener::bind(format!("127.0.0.1:{}", port))
            .await
            .expect("Failed to bind axum");
        println!("[siege] API server listening on http://127.0.0.1:{}", port);
        axum::serve(listener, app).await.expect("Axum server error");
    });

    // Wait for server to be ready
    tokio::time::sleep(std::time::Duration::from_millis(200)).await;

    tauri::Builder::default()
        .plugin(tauri_plugin_shell::init())
        .setup(move |app| {
            // Inject API_BASE into webview
            let window = app.get_webview_window("main").unwrap();
            let script = format!(
                "window.__SIEGE_API_BASE__ = 'http://127.0.0.1:{}';",
                port
            );
            window.eval(&script).ok();
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

use rusqlite::Connection;
use std::path::PathBuf;

pub fn get_data_dir() -> PathBuf {
    let base = dirs::data_dir()
        .unwrap_or_else(|| PathBuf::from("."));
    let dir = base.join("siege");
    std::fs::create_dir_all(&dir).ok();
    dir
}

pub fn init_db() -> Result<Connection, rusqlite::Error> {
    let db_path = get_data_dir().join("siege.db");
    let conn = Connection::open(&db_path)?;
    conn.execute_batch("PRAGMA journal_mode=WAL; PRAGMA foreign_keys=ON;")?;
    migrations::run(&conn)?;
    Ok(conn)
}
```

```typescript
/**
 * API base URL. In Tauri desktop mode, Rust injects window.__SIEGE_API_BASE__.
 * In web/dev mode, defaults to empty string (same-origin).
 */
function getApiBase(): string {
  if (typeof window !== "undefined" && (window as any).__SIEGE_API_BASE__) {
    return (window as any).__SIEGE_API_BASE__;
  }
  return "";
}

/**
 * Wrapper around fetch() that prepends API_BASE to relative URLs.
 * Drop-in replacement for fetch("/api/...").
 */
export function apiFetch(input: string | URL | Request, init?: RequestInit): Promise<Response> {
  if (typeof input === "string" && input.startsWith("/api/")) {
    return fetch(`${getApiBase()}${input}`, init);
  }
  return fetch(input, init);
}
```
